
# You'll need a YouTube API Key to get search results

1. You'll need a Google account first
2. Go to https://console.developers.google.com/apis/dashboard
3. Create Project
4. Create
5. Give it a name and Create
6. Click "View All" link
7. Search "Youtube" and select "YouTube Data API"
8. Enable
9. Click "Create Credemtials" button
10. Select "Web server"
11. Select "Public data"
12. Click "What credentials do I need?"
13. You should now have an API key for YouTube. Copy it and save it to a text file in the same directory with your bot code files. That text file should be named "YTKey.txt"
